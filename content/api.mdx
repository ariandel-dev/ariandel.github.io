## REST vs GraphQL

REST의 경우는 Relation을 payload에 같이 포함시킬 경우, Principle을 위반하게 됩니다.   
GraphQL은 Relation을 포함하는 것이죠.

GraphQL의 기본 개념은 API를 호출하는 Client에게 책임을 넘김으로 인해 좀 더 유연한 개발을 위함
