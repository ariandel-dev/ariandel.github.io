### Cluster

Aurora 클러스터 볼륨은 다중 가용 영역을 아우르는 가상 데이터베이스 스토리지 볼륨으로서, 각 가용 영역에는 DB 클러스터 데이터의 사본이 있습니다.

RDS 프록시는 애플리케이션 연결을 유지하면서 예비 DB 인스턴스에 자동으로 연결하여 데이터베이스 장애에 대한 애플리케이션의 복원력을 높입니다. 

### 특징

역시 aws 서비스답게 설정이 편리하다.  
db 생성후 elasticache 연동 또한 클릭 몇번으로 가능하다.  

인스턴스 성능 또한 in-place 업그레이드가 가능해서 일단 t4g medium으로 설정을 했다.   
인스턴스 종류에는 크게 r시리즈와 t시리즈가 있는데 r 시리즈가 메모리 최적화가 되어있는 인스턴스라
성능상의 이점은 있지만 기본 비용이 높다.
